# 🔔 Sistema de Notificações (Factory Method)

Projeto em Java que implementa envio de notificações usando o padrão Factory Method.

## 📚 Conceito
Permite criar diferentes tipos de notificações sem acoplamento direto.

## 📦 Tipos de Notificação
- Email
- SMS
- Push Notification

## 🏗️ Estrutura

br.fatec <br>
├── cliente <br>
├── factory <br>
└── model


## ▶️ Execução
```bash
java br.fatec.cliente.Main
```

📌 Padrão
Factory Method
